# test-vectors

Artifacts for the P1.1 platform-evaluation go/no-go gate (S2-D2 spec, step 6).
The HDF5/NetCDF files themselves are **not** committed here — only the
generator script and a pointer to the real NOAA sample.

## Synthetic 10 GB file

Generated by [`gen_synthetic.rs`](gen_synthetic.rs) via ClawHDF5's high-level
`FileBuilder` API. Produces three datasets at different chunk sizes
(`dataset_64kb`, `dataset_256kb`, `dataset_1mb`) so chunk-size-dependent
benchmarks (P1.2, P1.2b) have a single shared input file.

```bash
cargo run --example generate_10gb --release 10 /tmp/noaa_data/synthetic_10gb.h5
```
